DisplayXR OpenXR Runtime

This installer will set up the DisplayXR OpenXR runtime on your Mac.

Components:

  • DisplayXR Runtime — OpenXR runtime with Vulkan compositor and MoltenVK, registered as the system’s active OpenXR runtime.
  • SimCube OpenXR — A double-clickable test application that renders a spinning cube.

To uninstall, run the included uninstall.sh script from /Library/Application Support/DisplayXR/.
